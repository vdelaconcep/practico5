Trabajo práctico N°5

Ejercicio 1

En la consola se imprimirá una cantidad de gatos distinta cada vez, con 3 emojis diferentes que se irán intercalando.

Ejercicio 2

En la consola se imprimirá una cantidad de gatos distinta cada vez, cada uno con un número de huellas a su lado (el mismo número de huellas para cada gato, distinto cada vez)

Ejercicio 3

Tal como en el ejercicio 2, se imprimirá en la consola una cantidad de gatos distinta cada vez, cada uno con un número de huellas a su lado (el mismo número de huellas para cada gato, distinto cada vez), pero se alternará entre dos gatos de diferente color.
